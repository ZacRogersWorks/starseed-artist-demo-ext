# Starseed Artist Extension Demo
Showcasing the artist view
## How to Install and Run Locally
1. Download code from Github
2. Open terminal, cd to project directory and run the following commands:
    - `npm install`
    - `npm run build`
3. Open Google Chrome and navigate to `chrome://extensions/` 
4. Activate Developer Mode in upper right corner.
5. Click on "Load Unpacked" and select the `dist` directory from project directory
6. The extension is now ready to use, you can pin the extension by clicking on the puzzle icon in your browser


## Built using:
- React 18 with TypeScript
- Webpack 5
- Tailwind CSS
- PostCSS
